# Weiss

> Weiss is a 2D and 3D rendering engine for windows built with DirectX in c++. It is intended to be a wrapper around low-level apis to make the developpement of 2D and 3D applications simpler and faster.

## Getting Started

Clone the repository with [Git](https://git-scm.com/)

```
git clone "https://github.com/PolarToCartesian/Weiss"
```
